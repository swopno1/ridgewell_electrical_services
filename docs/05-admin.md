# Administrator Console

The **Admin Console** provides tools for managing the workforce, project portfolio, and financial reporting.

---

## 1. Employee Directory Management

Only users with the **ADMIN** role can register new employees or modify existing accounts.

### Registering New Employees
1. Navigate to **Employees** -> **Add Employee**.
2. Fill in: **Full Name**, **Email**, and **Role** (EMPLOYEE, MANAGER, ADMIN).
3. Click **Create Employee**.

#### Default Setup
- **Leave Balance**: Automatically initialized to **20 days** for the current year.
- **Initial Password**: New accounts use default: `Employee@123456`.
- **First Steps**: Employees should change this password immediately via **Settings**.

### Modifying Employee Details
1. Locate the employee and click **Edit** (pencil).
2. Update profile details or change the user role.
3. Click **Save Changes**.

---

## 2. Account Deactivation

Managing account status is critical for security.

> [!CAUTION]
> **Deactivation Impact**
> Setting an account to **Inactive** immediately:
> - Blocks the user from logging in.
> - Invalidates all active sessions.
> - Prevents new timesheets or leave requests.

---

## 3. Project Administration

Managers and Admins can manage the project list to ensure accurate time tracking.

### Creating New Projects
1. Navigate to **Projects** -> **Create Project**.
2. Provide **Project Name**, **Client Name**, and **Description**.
3. Set status to **Active** or **Inactive**.

### Managing Project Status
- **Active**: Visible in timesheet dropdowns.
- **Inactive**: Hidden from new entries but retained for historical reports.

---

## 4. Reports & Exports

TimesheetPro provides reporting tools to summarize hours for payroll and costing.

### Generating Summaries
1. Navigate to **Reports**.
2. Select report type: **Hours Summary**, **Payroll Summary**, or **Project Report**.
3. Use the **Date Range Picker** to filter results.

### Exporting Data
1. Click **Export** at the top of any report view.
2. Select **CSV** (for Excel/Payroll) or **PDF** (for Archiving).
